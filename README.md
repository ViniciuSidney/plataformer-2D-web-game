# 2D Plataformer 2D Web Game

Um jogo plataforma 2D minimalista desenvolvido com tecnologias web.

---

## 🎮 Sobre o projeto

Este projeto tem como objetivo criar um jogo plataforma 2D para navegador, utilizando HTML, CSS e JavaScript.

A proposta inicial é desenvolver uma base jogável simples, organizada e evolutiva, com foco em movimento, pulo, colisões, fases curtas e boa estrutura de código.

---

## 🧱 Tecnologias

- HTML
- CSS
- JavaScript
- Canvas API

---

## 🚧 Status

Projeto em fase inicial de planejamento e estruturação.

---

## 📌 Objetivos iniciais

- Criar a estrutura base do projeto
- Configurar o Canvas
- Implementar o loop principal do jogo
- Criar o jogador
- Adicionar movimento lateral
- Adicionar gravidade e pulo
- Implementar colisões simples com plataformas

---
