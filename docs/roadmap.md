# Roadmap

## v0.1 — Base inicial jogável

### Objetivo
Criar a primeira fundação jogável do projeto, com estrutura organizada, Canvas funcional, jogador controlável, câmera, física básica, colisão com plataformas, objetivo final, reinício e ferramentas iniciais de debug para criação de fases.

### Status
Concluída.

## Fechamento da v0.1

A versão v0.1 foi considerada concluída porque o jogo possui:

- Um jogador controlável
- Movimento lateral funcional
- Física refinada de pulo e queda
- Câmera acompanhando o jogador
- Plataformas com colisão completa
- Uma fase simples jogável
- Um objetivo final
- Condição de vitória
- Sistema de reinício
- Morte por queda
- Ferramentas básicas de debug e edição de fase

---

# v0.2 — Estrutura de jogo e experiência inicial

### Objetivo
Transformar a base jogável da v0.1 em uma experiência mais completa, com menu, fluxo de telas, sistema de fases, obstáculos, coletáveis e melhorias visuais minimalistas.

### Status
Planejada.

---

## Próximos passos da v0.2

- Criar menu inicial
- Criar tela de pausa
- Criar tela de vitória mais organizada
- Criar tela de derrota mais organizada
- Criar sistema de troca de fases
- Criar pelo menos uma segunda fase
- Adicionar obstáculos simples
- Criar entidade de perigo/dano
- Adicionar coletáveis simples
- Criar contador de coletáveis
- Melhorar visual minimalista das plataformas
- Melhorar visual do objetivo final
- Criar interface básica dentro do jogo
- Adicionar controle para ativar/desativar debug
- Separar melhor modo jogo e modo edição
- Melhorar documentação de criação de fases

---

## Possível fechamento da v0.2

A versão v0.2 será considerada concluída quando o jogo tiver:

- Menu inicial funcional
- Fluxo básico de telas
- Pelo menos duas fases jogáveis
- Sistema de troca de fases
- Obstáculos simples
- Coletáveis básicos
- Interface mínima de jogo
- Debug controlável
- Visual minimalista mais definido
