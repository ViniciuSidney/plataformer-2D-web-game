# Guia de Símbolos dos Mapas

## Entidades

| Símbolo | Significado |
|---|---|
| `.` | Vazio |
| `J` | Início do jogador |
| `G` | Portal / objetivo final |
| `C` | Moeda coletável |
| `H` | Perigo / espinho |

## Tiles sólidos

| Símbolo | Uso |
|---|---|
| `B` | Terreno/chão fundível |
| `P` | Plataforma conectável |
| `S` | Plataforma separada/destacada |

## Recomendações

Use `B` para chão, paredes, pilares e estruturas que devem parecer parte do terreno.

Use `P` para plataformas que podem se conectar visualmente com outras plataformas.

Use `S` para peças que devem parecer independentes, com mais destaque visual.
